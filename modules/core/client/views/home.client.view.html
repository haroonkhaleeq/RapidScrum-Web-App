Rapid Scrum

Introduction

This project is developed as a requirement of the subject "Internet - Practical Course Telecooperation". This course was offered in winter semester 2015-2016 in Informatik department of TU Darmstadt. The main purpose of this course was to "Learn how to develop modern Web Applications". The project work has a team structure consisting of 3-5 members. Our team consists of 5 members. The genaral requirement of the project was to develop a web application for Managing Agile Projects, using MEAN (MongoDB, Express, Angular, NodeJS) Stack.

Azmat Ali Khan comments about this project
Scrum is an agile methodology that can be applied to nearly any project; however, the Scrum methodology is most commonly used in software development. The Scrum process is suited for projects with rapidly changing or highly emergent requirements. Scrum software development progresses via a series of iterations called sprints, which last from one to four weeks. The Scrum model suggests each sprint begins with a brief planning meeting and concludes with a review. These are the basics of Scrum project management.
